第 2 章：Java 程序设计环境(Win 10)

  1. 安装 JDK (Java SE Development Kit)

  2. 进入Environment Variables ，在 System variables 中新建变量，变量名为Java_Home ，变量值为C:\Java\jdk1.8.0_151(安装路径) 。

  3. 选择 System variables 的 Path 变量，添加 %Java_Home%\bin 和 %Java_Home%\jre\bin 两条值。
