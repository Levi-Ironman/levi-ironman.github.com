eNSP 实验总结

发表于 2019-03-25 |

（一）eNSP 及 VRP 基础操作

在用户视图下只能使用参观和监控级命令：

display version 显示系统软件版本及硬件等信息。

system-view 切换到系统视图。

quit 切换回用户视图。

interface 命令进入接口视图: interface GigabitEthernet 0/0/0 。

ip address 命令配置接口IP 地址，子网掩码。

return 直接退回到用户视图。

命令行帮助：? 获取该命令视图下所有命令

interface   ?   列出全部关键字

inter?    列出以该字符串开头的所有关键字

在系统视图下，使用 sysname R1 命令修改当前路由器名称.

display current-configuration 查看路由器当前配置。

display interface GigabitEthernet 0/0/0 查看该接口的状态信息。

display ip interface brief 查看接口与IP相关摘要信息。

display ip routing-table 查看路由表信息。

（二） 交换机基础配置

  1. 交换机基础配置

    配置交换机双工模式：配置接口的双工模式可在自协商或非自协商模式下进行。

    配置接口速率： 在自协商模式下，以太网接口的速率是和对端接口协商得到的，在非自协商模式下，需手动配置。

  2. 理解ARP 及 Proxy ARP

    ARP (Address Resolution Protocol) 是用来将 IP 地址解析为MAC地址的协议 。ARP 表项可以分为动态和静态两种类型。设备上有一个ARP 高速缓存，用来存放IP地址到MAC地址的映射表，利用ARP 请求和应答报文来缓存映射表。

    Proxy ARP 代理ARP ， 当主机上没有配置默认网关地址，可以发送一个广播ARP请求，请求目的主机的mac地址，使具备Proxy ARP 功能的路由器收到这样的请求后，在确认请求地址可达后，会使用自身的mac地址作为该ARP 请求的回应，使得处于不同物理网络的同一网段的主机之间可以正常通信。

    arp -a 查看主机的ARP 表

    display arp all 查看路由器ARP表

    arp static 10.1.1.1 1111-2222-3333 配置静态ARP

    arp-proxy enable 开启路由器接口ARP代理功能

（三） VLAN

  1. Vlan 基础配置及Access 接口

    Access 接口是交换机上用来连接用户主机的接口，当Access接口从主机收到一个不带VLAN标签的数据帧是，会给该数据帧加上与PVID一致的VLAN标签，当Access 接口要发送一个带VLAN标签的数据帧给主机时，首先检查该数据帧的VLAN ID 是否与自己的PVID 相同，若相同，则去掉VLAN标签后发送该数据帧给主机，若不同，则丢弃该数据帧。

    vlan 10 创建VLAN 10

    vlan batch 20 30 创建VLAN 20 和 30

    display vlan 查看VLAN 信息

    port link-type access 配置接口为Access 接口

    port default vlan 10 配置接口默认VLAN 并加入VLAN 10

  2. 配置trunk接口

    Trunk 接口一般用于交换机之间连接的端口，可以属于多个VLAN。

    当Trunk 端口收到数据帧时，如果该帧不包含802.1Q的VLAN 标签，将打上该Trunk端口的PVID；如果该帧包含VLAN 标签，则不改变。

    当Trunk 端口发送数据帧是，当该所发帧的VLAN ID 与端口的PVID 不同时，检查是否允许该VLAN通过，若允许的话直接透传，不允许就直接丢弃；当该帧的VLAN ID与端口的PVID相同时，则剥离VLAN标签后转发。

    port link-type trunk 配置接口为trunk接口，允许VLAN 10 和VLAN 20 通过

    port trunk allow-pass vlan 10 20

  3. 理解Hybrid接口的应用

    Hybrid 接口即可以连接普通终端的接入链路又可以连接交换机间的干道链路，它允许多个VLAN的帧通过，并可以在出接口方向将某些VLAN帧的标签剥掉。

    port link-type hybrid 配置接口为默认的Hybrid 类型。

    port hybrid untagged vlan 20 以untagged 的方式发送给PC。

    port hybrid pvid vlan 20 设置接口默认VLAN ID 。

    port hybrid tagged vlan 10 20 设置仅能通过VLAN 10 和VLAN 20的数据帧。
